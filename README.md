**🎵 ESP32 MUSIC PLAYER 🎵**

***Componentes usados en el proyecto.***

📶 ESP32 [1].

⚡️ Resistencia [1]. De 1 kΩ.
💡 LED 04. De color verde. Indicador de instanciación/reposo.

💡 LED 18. De color rojo. Indicador de ejecución de canciones.
🔊 Buzzer pasivo (piezo o zumbador).

🔌 Conexiones [5].
